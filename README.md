### Compile from source

To compile the app from source you'll need git and a recent version of Node.js (see [Contributing](./CONTRIBUTING.md))
for more details.

```shell
$ git clone https://github.com/polito-it/students-app.git
$ cd students-app
$ npm install
$ npx expo build:[android|ios]
```
